**Features:**
- Silences player voice types (Brave, Clever, Gruff, Weird)

**Installation:**

**Prerequisites:**
1. Install PythonSDK Mod Manager
   - Download from: https://github.com/bl-sdk/oak-mod-manager/releases (grab the Wonderlands version)
   - Extract and copy to your game directory:
     - Steam: `Steam\steamapps\common\TinyTinaWonderlands`
     - Epic: `Program Files\Epic Games\TinyTinasWonderlands`

2. Install OpenHotfixLoader
   - Download from: https://github.com/apple1417/OpenHotfixLoader/releases
   - Extract and copy to: `OakGame\Binaries\Win64\Plugins`

**Mod Installation:**
1. Download this mod file
2. Place it in your mod directory
